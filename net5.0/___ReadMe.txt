How to use:

Run this with a windows CMD (others like powershell dont seem to work), like so:

NOTE: The X & Y coords are for the TOP LEFT COORDINATE
The width & height values are in PIXEL COUNT

PlaceGridConverter.exe [filepath&name] [xcoord] [ycoord] [width] [height]

so:

PlaceGridConverter.exe C:\Users\Kunkelbek\Documents\mona_lisa.jpg 600 500
